# FitTrack Mobile App 📱

A cross-platform mobile application for fitness, nutrition, and health tracking, developed with Flutter. It offers a modern user experience with integrated Artificial Intelligence to maximize training results.

## 🌟 Features
- **Dynamic Dashboard:** Clear visualization of weekly progress and daily goals.
- **Routine Planner:** Intuitive workout builder with a comprehensive exercise library.
- **Nutritional Logging:** Scan and log meals with detailed macro breakdowns.
- **AI Coach:** Integrated chatbot for real-time assistance and automatic routine generation.
- **Data Visualization:** Interactive charts for performance and body metrics.

## 🛠️ Tech Stack
- **Language:** Dart
- **Framework:** Flutter
- **State Management:** Provider / Bloc
- **Communication:** REST API with Node.js backend
- **Design:** Material Design 3 with Clean Architecture principles.

## 🚀 Quick Start Guide

1. **Requirements:** Flutter SDK installed.
2. **Get dependencies:**
   ```bash
   flutter pub get
   ```
3. **Configure API:** 
   Adjust the server URL in `lib/globals/global_variables.dart`.
4. **Run:**
   ```bash
   flutter run
   ```
